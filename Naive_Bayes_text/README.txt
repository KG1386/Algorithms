Naive Bayes implementation for sorting the text files into positive and negative reviews. Uses nltk library for stemming (increases accuracy). 
For training, a set of text files with positive and negative sentences has been used, taken from here:
sentence polarity dataset v1.0
http://www.cs.cornell.edu/People/pabo/movie-review-data/
Currently has 70% accuracy on reviews from metacritic(test has been done on further ends of user score - below 30 and higher than 70)
